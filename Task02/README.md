Веб-версия консольной игры "Наибольший общий делитель".

Требования
- PHP 7.4 или выше
- SQLite3
- Composer (опционально)

Установка и запуск
1. Клонирование репозитория
2. Создание базы данных
cd db
php init_db.php
3. Запуск веб-сервера
php -S localhost:3000 -t public
4. Открыть в браузере
Перейдите по адресу: http://localhost:3000

Правила игры
Вам будет задано 3 вопроса
Найдите наибольший общий делитель (НОД) двух чисел
За каждый правильный ответ начисляется 1 балл
Игра продолжается до конца, даже если были ошибки

Структура проекта
public/index.php — главная страница
public/game.php — страница игры
public/result.php — страница результата
db/games.db — БД SQLite
db/init_db.php — скрипт инициализации БД
README.md — Приложение с информацией о проекте
